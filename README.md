# Flash
## Введение

флеш сообщения

## Использование

установка ключа и сообщения 
```php
Flash::set('error', 'Error Message');
```
получение сообщения с помощью переданного ключа 
```php
Flash::get('error');
```
удаление указанного ключа 
```php
Flash::del('error');
```
проверка на существование ключа
```php
Flash::exists('error');
